## Installation

- Fork the Repository and clone it at the desired location

```bash
  git clone https://github.com/Deepak91168/property-hunt.git
```

- Open Repository

```bash
 cd property-hunt
```

- Install required dependencies

```bash
 npm install
```

- Start Application at your localhost

```bash
 npm start
```
